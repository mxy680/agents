## How This Works

- **Learning jobs** run every 2 hours, scanning recent emails and texts for significant information
- **Interactive sessions** update memories when Mark shares something meaningful in conversation
- Each entry is dated so stale information can be identified and retired
- Raw email/text content is never stored — only summaries and patterns
